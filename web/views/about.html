Projeto SIG-TRAC 2016

Projeto Interdisciplinar SIG-TRAC ( Sistema I ntegrado de   Gerenciamento em  Tempo  Real de  Alertas e  C rises )

Este portal tem por objetivo centralizar os materiais de aulas, listas de exercícios, relações de alunos e artefatos produzidos no IPBL (Interdisciplinary Problem-Based Learning), desenvolvidos durante o 2º Semestre de 2016, envolvendo as seguintes disciplinas:
  • CES-65 - Projeto de Sistemas Embarcados
  • CE-235 -  Sistemas Embarcados de Tempo Real
  • CE-230  -  Qualidade, Confiabilidade e Segurança (Safety) de Software
  • CE-237 -  Tópicos Avançados em Teste de Software
Todo o material publicado neste portal deve ser utilizado somente para propósitos acadêmicos.
